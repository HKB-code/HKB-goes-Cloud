HKB-goes-Cloud ☁️
From zero to cloud-native — learning Go, distributed systems, and infrastructure engineering through hands-on projects, one commit at a time.

What This Repo Is 🗄️
This is my public learning journal as I build the skills to become an AI Backend / Cloud Infrastructure Engineer. Every folder here represents real work — not tutorials copy-pasted, but concepts understood and rebuilt from scratch.
The goal isn't just to learn Go or pass a certification. It's to develop systems thinking — the ability to look at a problem and reason about concurrency, fault tolerance, scaling, and trade-offs before writing a single line of code.

How I Learn 🥸

1. Watch / Read — Absorb the concept from courses or docs
2. Rebuild from memory — Close the tutorial and write it again blind
3. Break it — Intentionally introduce bugs to understand failure modes
4. Document it — If I can't explain it simply, I don't understand it yet
5. Commit daily — Consistency over intensity
